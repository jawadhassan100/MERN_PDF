# MERN Form To PDF <br/>
# > This app is fully built on MERN Stack <br/>
# > It take data in form and save it in card then user can delete it if want 
# > And User can download that form as a pdf
<hr/>

# Technologies
. React <br/>
. Node  <br/>
. Express js <br/>
. MongoDB <br/> 
. Tailwind CSS <br/>
. pdfKit
